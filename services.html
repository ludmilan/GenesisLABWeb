Hotel Food Reservation.

Hotel Food Reservation is an Android app which helps the hotel/restaurant management to ease the process of food delivering and for customers to reserve food at hotels/restaurants. Through this app a customer can order foods, set their preferred meals as favorites, rate the service, and easily pay the bill for the respective meal ordered.

QA Automation Framework.

This QA Automation framework was developed on Java using the Selenium Web Driver and TestNG frameworks along with the Page Object Model. The framework makes it easier even for a tester without technical knowledge to do the automation stuff. Easy to use and ultimately provides detailed Reports on the Automation Test Results using TestNG. And the framework has the capability of automating the Desktop applications as well as Mobile applications with integration of the SikuliX Automation scripts.

Real Time Car Racing.

Android Car Racing Game syncing with custom built toy cars. This is kind of a mobile game with physical experience. A Toy Car is remotely handled via an android app using bluetooth where the Toy car is equipped with Arduino technology. The player is facilitated with a live video stream from the car using a wifi connection between the app and the toy car. A specific track is assigned for the Race where the track has hot spots to pass through. When a hotspot is passed right beneath the player gets a speed boost. Game has its own rules. This Mobile game is hosted for gaming contests on appeal.
